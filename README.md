# Bill-Calculator-in-python
This is a Python script designed to automate the process of splitting a restaurant bill. It calculates the total cost including a tip and divides the final amount equally among a group of people.

How to Use:
Ensure you have Python 3.x installed on your system.
Clone this repository or download the bill_split.py file.
Open your terminal or command prompt.
Run the script using the following command:
Bash
python bill_split.py

Follow the on-screen prompts to enter the bill amount, tip percentage, and number of people.
